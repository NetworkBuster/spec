# Code of Conduct

This project adopts the Contributor Covenant v2.1 as its Code of Conduct.

Be respectful, inclusive, and professional in all discussions and contributions.

If you experience or witness unacceptable behavior, please contact the repository owner: andrew@networkbuster.net.

Full text and enforcement policy should be considered part of this repository's governance.
